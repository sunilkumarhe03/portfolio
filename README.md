# Sunil Kumar H.E | Portfolio

Professional portfolio of Sunil Kumar H.E, an Electronics and Communication Engineer specializing in Embedded Systems, Python, and IoT.

## Local Development

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key (if needed for AI features)
3. Run the app:
   `npm run dev`
